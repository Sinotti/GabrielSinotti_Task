## Controls

### Gameplay

- **W**: Walk Forward
- **S**: Walk Backward
- **A**: Walk Left
- **D**: Walk Right
- **Left Shift**: Run
- **E**: Interact

### UI

- **Tab**: Inventory
- **Esc**: Pause
